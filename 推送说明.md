## GitHub Pages 部署

如果需要在 GitHub Pages 上部署网站，请按以下步骤操作：

### 1. 启用 GitHub Pages

1. 访问：https://github.com/de-bao/pages/settings/pages
2. 在 "Source" 部分：
   - 选择分支：`main`
   - 选择文件夹：`/ (root)` 或 `/pages`（根据实际情况）
3. 点击 "Save" 保存

### 2. 访问网站

部署完成后（通常需要几分钟），可以访问：
- **网站地址**：`https://de-bao.github.io/pages/`

如果需要使用自定义域名，可以在仓库设置中配置。

## 故障排除

### 推送失败

如果推送失败，请检查：

1. **网络连接**：确保可以访问 GitHub
2. **认证信息**：检查 GitHub token 是否有效
3. **仓库权限**：确保有推送权限
4. **分支冲突**：如果远程有新的提交，需要先拉取：
   ```bash
   git pull origin main --rebase
   git push origin main
   ```

